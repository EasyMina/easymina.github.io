# consoleLanguage

Here, you can set the language of the terminal output.


## Specification

Default implementation: ```{ "consoleLanguage": "en" }```

### Key

| **Name** | **Alias** | **Category** |  
|:--|:--|:--|
| ```consoleLanguage``` | ```messages__use``` | [Default](../options/#default) |

### Value

Currently only "en" is valid

| **Default** | **Validation** | **Type** |
|:--|:--|:--|
| ```en``` | ```^(en|none)$``` | ```string``` |
